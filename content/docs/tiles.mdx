---
title: 地图切片服务
description: 矢量切片、缓存策略和缩放级别建议。
---

在线地图建议优先使用矢量切片。矢量切片体积小、样式灵活、适合多端展示。影像和复杂栅格数据使用栅格切片或云优化 GeoTIFF。

:::tip
热点区域预生成切片，低频图层按需生成并缓存；所有切片 URL 加版本号，便于灰度发布和缓存失效。
:::

```http
GET /tiles/{layer}/{z}/{x}/{y}.pbf
Cache-Control: public, max-age=86400, stale-while-revalidate=604800
```

## 缩放级别建议

| 缩放级别 | 展示内容 | 处理策略 |
| --- | --- | --- |
| z0 - z8 | 行政区、主干道路、聚合指标 | 只输出概览数据，避免加载明细要素 |
| z9 - z13 | 街道、园区、业务区域 | 输出简化几何和核心属性 |
| z14+ | 地块、资产点、管线节点 | 输出高精度几何，并按权限过滤敏感字段 |
