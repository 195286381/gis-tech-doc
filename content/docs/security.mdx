---
title: 安全与权限
description: 接口访问、图层权限、空间权限和数据脱敏。
---

GIS 数据常包含敏感位置、资产和业务边界。权限模型需要同时覆盖接口访问、图层可见性、空间范围和导出能力。

| 控制点 | 要求 |
| --- | --- |
| 身份认证 | 统一接入 SSO、OAuth 或企业身份系统 |
| 图层权限 | 按角色控制图层列表、字段可见性和样式配置 |
| 空间权限 | 按行政区、项目范围或自定义多边形限制查询结果 |
| 数据脱敏 | 敏感点位可做坐标偏移、聚合展示或最小缩放级别限制 |

## 安全基线

- 生产环境 API 域名、地图 Token 和图层配置不得写入源码。
- 所有导出任务必须记录操作者、导出范围、字段列表和审批结果。
- 高敏感图层默认关闭公共访问，并限制最小缩放级别。
